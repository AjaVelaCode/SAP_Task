The code was done in IntelliJ IDEA 2019.3.5.
From IntelliJ IDEA, it is possible to run main program (Run 'Main.main()') and all tests (Run 'All tests') successfully.

From command prompt
- To compile program from command prompt: reach final folder and type javac *.java  
- To run project from command prompt: reach src folder and type java com.company.Main 
- To compile test from command prompt: reach final folder and type javac -cp C:\...\src\;C:\...\lib\* *.java
- To run test from command prompt: reach final folder and type java -cp "C:\...\src\;C:\Program Files\JetBrains\IntelliJ IDEA Community Edition 2019.3.4\lib\junit-4.12.jar" org.junit.runner.JUnitCore (test)
